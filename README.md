## Recommended to use Java 21

## Function keys
- **H**: Toggle full gamma (full brightness)
- **J**: Shader menu
- **K**: Toggle sky
- **Z**: Zoom
- **V**: Voice chat settings
- **M**: Mute voice chat microphone
